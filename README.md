# 🔍 About the Project

This project focuses on predicting whether a customer will accept a personal loan offer based on their financial and demographic data. The goal is to help banks identify potential customers and improve marketing strategies.

# 🎯 Objective
Predict loan acceptance using customer data
Identify key factors influencing loan decisions
Build models to support data-driven marketing

# 📊 Dataset Overview

The dataset contains customer details such as:
Age, Income, and Experience\
Credit card usage and spending\
Mortgage and account details\
Online banking activity\
Only a small percentage of customers accept loans, making this a classification problem with imbalanced data

# ⚙️ Work Done
Data cleaning and preprocessing\
Exploratory Data Analysis (EDA)\
Feature selection\
Model building and evaluation

# 🤖 Models Used
Logistic Regression\
Decision Tree\
Random Forest\
K-Nearest Neighbors

# 📈 Key Insights
Income and spending patterns strongly influence loan acceptance\
Customers with certain account types are more likely to accept loans\
Data imbalance plays a major role in model performance
